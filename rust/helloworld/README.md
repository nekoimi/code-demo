# Hello World

### 初始化新项目

```shell
cargo new helloworld
```

### 运行项目

```shell
cargo run

or

cargo run --release
```

- 编译 （debug）

```shell
cargo build
```

or

- 编译 （release）

```shell
cargo build --release
```

- 运行

```shell
.\target\debug\helloworld.exe
```


### 快速代码检查

```shell
cargo check
```
